# 要求仕様書

## 概要

指定した時間からカウントダウンを行い、時間経過を視覚的に表示するタイマー機能を提供する。

## 目的

- ユーザーが特定の時間を設定し、その時間が経過するまでのカウントダウンを行えるようにする
- 残り時間を分かりやすく表示し、時間管理を支援する
- タイマー終了時に通知を行い、ユーザーに時間経過を確実に伝える

## ストーリー

* 一般ユーザー: 日常的に時間管理が必要な人
  * カップ麺を作る際に3分タイマーをセットして、出来上がりを待つ
  * 集中作業のために25分のポモドーロタイマーをセットして作業する
  * 休憩時間を管理するために5分タイマーをセットする
* 料理をする人: キッチンで調理時間を管理したい人
  * オーブンの調理時間をタイマーで管理する
  * 複数の調理工程を並行して管理するために複数のタイマーを使う

## 機能要件

### REQ-1: タイマー設定

| 要件ID | 要件内容 | 優先度 | 備考 |
|--------|----------|--------|------|
| REQ-1.1 | ユーザーは時間（時・分・秒）を指定してタイマーを設定できる | MUST | 最大99時間59分59秒まで設定可能 |
| REQ-1.2 | よく使うプリセット時間（1分、3分、5分、10分、25分）を選択できる | SHOULD | ワンタップで設定可能 |
| REQ-1.3 | 過去に使用したタイマー時間の履歴から選択できる | MAY | 直近10件を保存 |

### REQ-2: タイマー操作

| 要件ID | 要件内容 | 優先度 | 備考 |
|--------|----------|--------|------|
| REQ-2.1 | タイマーを開始できる | MUST | |
| REQ-2.2 | タイマーを一時停止できる | MUST | |
| REQ-2.3 | 一時停止中のタイマーを再開できる | MUST | |
| REQ-2.4 | タイマーをリセット（初期設定時間に戻す）できる | MUST | |
| REQ-2.5 | タイマーをキャンセル（停止して設定画面に戻る）できる | SHOULD | |

### REQ-3: タイマー表示

| 要件ID | 要件内容 | 優先度 | 備考 |
|--------|----------|--------|------|
| REQ-3.1 | 残り時間を時:分:秒形式で表示する | MUST | 大きく見やすいフォント |
| REQ-3.2 | 進捗を視覚的に表示する（プログレスバーまたは円形インジケーター） | SHOULD | |
| REQ-3.3 | 残り時間が少なくなったら警告色で表示する | MAY | 残り10秒以下で赤色表示 |

### REQ-4: タイマー終了通知

| 要件ID | 要件内容 | 優先度 | 備考 |
|--------|----------|--------|------|
| REQ-4.1 | タイマー終了時に音声アラームを鳴らす | MUST | |
| REQ-4.2 | タイマー終了時に画面表示で通知する | MUST | |
| REQ-4.3 | アラーム音を選択できる | MAY | 複数の音声から選択 |
| REQ-4.4 | 通知をタップまたはクリックで停止できる | MUST | |

## 非機能要件

### NFR-1: ユーザビリティ

| 要件ID | 要件内容 | 優先度 | 備考 |
|--------|----------|--------|------|
| NFR-1.1 | 3クリック以内でタイマーを開始できる | SHOULD | 簡単な操作性 |
| NFR-1.2 | 画面を見なくても残り時間が分かる（音声読み上げ対応） | MAY | アクセシビリティ対応 |

### NFR-2: 信頼性

| 要件ID | 要件内容 | 優先度 | 備考 |
|--------|----------|--------|------|
| NFR-2.1 | タイマーの精度は±1秒以内とする | MUST | |
| NFR-2.2 | アプリがバックグラウンドでもタイマーが動作し続ける | SHOULD | 対応プラットフォームに依存 |
| NFR-2.3 | 端末がスリープ状態でもアラームが鳴る | SHOULD | 対応プラットフォームに依存 |

### NFR-3: パフォーマンス

| 要件ID | 要件内容 | 優先度 | 備考 |
|--------|----------|--------|------|
| NFR-3.1 | タイマー動作中のバッテリー消費を最小限に抑える | SHOULD | |
| NFR-3.2 | タイマー画面の表示更新は1秒間隔で行う | MUST | |
