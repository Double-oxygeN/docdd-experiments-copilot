# カウントダウンタイマーアプリ 機能要件仕様書

## 1. 概要

### 1.1 目的
ユーザーが指定した時間からカウントダウンを行い、時間経過を視覚的に表示するタイマーアプリケーション。

### 1.2 対象ユーザー
- 料理中の時間管理を行いたいユーザー
- 作業時間を区切りたいユーザー（ポモドーロテクニックなど）
- 運動やトレーニングの時間管理を行いたいユーザー
- 一般的な時間制限のあるタスクを管理したいユーザー

---

## 2. 機能要件

### 2.1 タイマー設定機能

| 項目 | 説明 |
|------|------|
| 時間入力 | 時・分・秒を個別に設定可能 |
| 入力範囲 | 00:00:01 〜 99:59:59 |
| プリセット | よく使う時間をプリセットとして保存（最大10件） |
| クイック設定 | 1分、5分、10分、15分、30分、1時間のワンタップ設定 |

### 2.2 タイマー操作機能

| 操作 | 説明 |
|------|------|
| 開始 | カウントダウンを開始する |
| 一時停止 | カウントダウンを一時的に停止する |
| 再開 | 一時停止したタイマーを再開する |
| リセット | タイマーを初期状態に戻す |
| キャンセル | 実行中のタイマーを中止する |

### 2.3 表示機能

- **残り時間表示**: 大きく見やすいデジタル表示（HH:MM:SS形式）
- **プログレスバー/円グラフ**: 経過時間の視覚的表示
- **進捗率表示**: パーセンテージでの進捗表示（オプション）
- **画面常時点灯**: タイマー実行中は画面がオフにならないオプション

### 2.4 通知・アラート機能

| 機能 | 説明 |
|------|------|
| 終了アラーム | タイマー完了時にアラーム音を再生 |
| バイブレーション | タイマー完了時に振動で通知 |
| プッシュ通知 | バックグラウンド動作時も通知を表示 |
| 事前通知 | 残り1分、30秒などで事前通知（設定可能） |
| アラーム音選択 | 複数のアラーム音から選択可能 |
| 音量設定 | アラーム音量の個別設定 |

### 2.5 複数タイマー機能

- 同時に複数のタイマーを実行可能（最大5件）
- 各タイマーに名前/ラベルを設定可能
- タイマーリストでの一覧表示

### 2.6 履歴機能

- 過去に使用したタイマー設定の履歴保存
- 履歴からのワンタップ再利用
- 履歴の削除機能

---

## 3. 非機能要件

### 3.1 パフォーマンス

- タイマーの精度: ±1秒以内
- アプリ起動時間: 2秒以内
- バッテリー消費: 最小限に抑える

### 3.2 信頼性

- バックグラウンド動作時も正確にカウントダウン継続
- アプリ強制終了後も再起動時にタイマー状態を復元
- オフライン環境でも全機能が動作

### 3.3 ユーザビリティ

- 直感的なUI/UXデザイン
- アクセシビリティ対応（スクリーンリーダー対応、高コントラストモード）
- ダークモード/ライトモード対応
- 多言語対応（日本語・英語を最低限サポート）

### 3.4 セキュリティ

- ユーザーデータはローカルに保存
- 不要な権限は要求しない

## 4. 画面構成

### 4.1 メイン画面
- タイマー表示エリア（中央・大）
- 操作ボタン（開始/一時停止/リセット）
- プリセット・クイック設定ボタン

### 4.2 設定画面
- アラーム音設定
- 通知設定
- 表示設定（テーマ、画面常時点灯など）
- プリセット管理

### 4.3 タイマーリスト画面
- 実行中・設定済みタイマーの一覧
- タイマーの追加・編集・削除

## 6. 技術仕様

### 6.1 対応プラットフォーム
- Web（PWA対応）
- iOS（iOS 14以上）
- Android（Android 8.0以上）

### 6.2 推奨技術スタック
- **フロントエンド**: React / React Native / Flutter
- **状態管理**: Redux / Context API / Provider
- **ローカルストレージ**: AsyncStorage / SharedPreferences / IndexedDB
- **通知**: Push Notification API / Firebase Cloud Messaging

## 7. 用語定義

| 用語 | 定義 |
|------|------|
| プリセット | ユーザーが事前に保存した時間設定 |
| クイック設定 | アプリに標準搭載されたよく使う時間設定 |
| インターバルタイマー | 複数のタイマーを連続して実行する機能 |
